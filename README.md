  ## Jarvis 3D GestureControl

A lightweight single-page application showcasing a 3D viewer and interactive UI for Jarvis-style scenes and assets.
Entry point: index.html with all necessary static files and models.

Live Demo: https://vrdeep101.github.io/3D-Jarvis-GCtrl/

## Features
* Lightweight static SPA built with HTML, CSS, and JavaScript
* Supports custom 3D models: .gltf and .glb formats


## 🎯 Purpose
* Demonstrate 3D model rendering and interactive UI on the web
* Provide a clean minimal app to extend for 3D web projects
* Learn and showcase web-based 3D technologies
* Building simple, interactive 3D web experiences with ease.
